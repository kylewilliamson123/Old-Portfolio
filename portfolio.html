JavaScript Weather APP

This is a javascript dominant application that uses Bootstrap and CSS to style. The application uses a fetch method to fetch data from the OpenWeatherMap API. Doing this has taught me about Javascript fetch and how to use APIs in an application. The user has the ablilty to search any city or zipcode in the world. The data that is retrieved and displayed is highs and lows for tempature, heat index, wind speed, wind direction, and the sky quality.
